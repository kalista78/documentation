## Image Hosting Tips

For logos, you can use:
- **Imgur**: Upload to imgur.com and copy direct link
- **IPFS**: Decentralized storage via Pinata or similar
- **Direct URLs**: Any publicly accessible image URL
- **Twitter**: Upload image in your tweet (bot will extract it)

<Warning>
  Image must be publicly accessible. Private or authenticated URLs won't work.
</Warning>

## Common Issues

<AccordionGroup>
  <Accordion title="Bot didn't respond">
    - Make sure you're following @zyperbot
    - Check your tweet format is correct
    - Ensure you haven't hit the 2 token daily limit
    - Try again in a few minutes
  </Accordion>
  
  <Accordion title="Invalid symbol error">
    - Symbol must be 3-10 characters
    - Use only letters and numbers
    - Don't include special characters (except $)
    - Check if symbol is already taken
  </Accordion>
  
  <Accordion title="Logo not showing">
    - Ensure URL is publicly accessible
    - Use direct image link (ends in .png, .jpg, etc)
    - Try re-hosting on Imgur
    - Image might be too large (keep under 2MB)
  </Accordion>
</AccordionGroup>
